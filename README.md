# FRE Assets

Deliverables, dashboards, reports, and tools built for FRE and TPB.

**Live directory:** [fre-work-directory.netlify.app](https://fre-work-directory.netlify.app)

## Structure
- `/dashboards/` — Interactive HTML dashboards
- `/decks/` — PowerPoint presentations
- `/reports/` — Briefs, memos, analysis docs
- `/research/` — Strategy docs, playbooks, market research
- `/data/` — Spreadsheets, CSVs, data files
- `/viz/` — Static visualizations (PNG)
- `/ntw/` — NTW / Project XX brand materials
- `/tko/` — TKO Properties & Partnerships
- `/tools/` — AI & tooling assets

## Access
All HTML files are browsable via GitHub Pages.  
Non-HTML files (PPTX, XLSX, PDF) download directly via raw links.
